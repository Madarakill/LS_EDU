## Описание .ipynb
- В файле Анализ набора данных.ipynb выполнен анализ данных и подготовка для нейронной сети.
- В файле Модель нейронной сети.ipynb выполнено проектирование архитектуры нейронной сети, обучение и проверка на тестовом наборе данных.

В рамках работы выполнены:
---
- анализ и предобработка данных;
- разработка архитектуры нейронной сети LeadMLP;
- обучение, валидация и тестирование модели;
- сохранение модели для дальнейшего использования. 

## Модель LeadMLP
Представляет собой многослойный перцептрон, принимающий на вход предобработанные табличные данные.


Система решает задачу лид-скоринга, присваивая клиенту бинарный класс:
---
- 1 — высокая вероятность конверсии,  
- 0 — низкая вероятность конверсии.
